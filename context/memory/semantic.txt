# Semantic Memory

Domain Concepts
- Monorepo managed by Nx; apps in `apps/`, libs in `libs/`.
- Services communicate over RabbitMQ; API Gateway (NestJS) + Frontend (Next.js).

Architectural Rationale
- Enforce boundaries with Nx tags and `@bakewise/*` path aliases.
- Unified error shape: { status, code, message, details? }.
- Soft-delete policy across entities; UUID v4 IDs.
